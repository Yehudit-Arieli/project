# משימת הערכה - פיתוח Backend (Node.js)

פרויקט זה נכתב כחלק ממשימת הערכה הבוחנת מידול נתונים, לוגיקה עסקית ועבודה עם Docker.

## דרישות המערכת שיושמו:
- [cite_start]**מידול נתונים:** שימוש ב-Prisma עם מודלים עבור `User`, `Game` ו-`GameParticipant` (קשרי Many-to-Many)[cite: 8, 9, 10, 11, 12].
- [cite_start]**לוגיקה עסקית:** מימוש פונקציית `joinGame` הכוללת וולידציה על סטטוס המשחק ובדיקת כפילות משתמשים[cite: 17, 19, 20, 21].
- [cite_start]**סביבת Docker:** הרצה מלאה של האפליקציה ובסיס הנתונים (PostgreSQL) בתוך קונטיינרים[cite: 30, 32, 33, 34].
